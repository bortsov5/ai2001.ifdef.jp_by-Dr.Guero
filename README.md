Основные компоненты системы:
1. Аппаратная часть:
Датчик BNO055 - 9-осевой IMU (инерциальный измерительный модуль) для определения ориентации

Сервоприводы ICS - управление суставами робота

Микроконтроллер - основной процессор (вероятно ARM)

Интерфейсы: I2C, UART, PIO, ADC/DAC

2. Кинематика робота:
Робот имеет две ноги с суставами:

K0, K1, K2 - суставы ног (колени)

U0, U1, U2 - суставы верхней части тела

A0, A1 - суставы рук

H - тазобедренные суставы

E - дополнительные суставы

3. Основные алгоритмы:
UVC (Universal Vestibular Control)
c
void uvc(void)
Стабилизация робота на основе данных IMU

Компенсация наклонов (pitch/roll)

Расчет необходимых перемещений ног для сохранения баланса

Управление походкой:
c
void walk(void)
Реализует конечный автомат с состояниями:

710 - Инициализация и сброс

720 - Калибровка датчиков ориентации

730 - Определение нейтрального положения

740 - Старт UVC контроля

750 - Подготовка к шагу

760 - Выполнение шага

770 - Завершение шага

Прямая кинематика:
c
void footCont(float x,float y,float h,int s)
Рассчитывает углы суставов для позиционирования стопы в пространстве.

4. Ключевые особенности:
Автоматическая балансировка - использует данные гироскопа и акселерометра

Адаптивная походка - подстраивается под наклоны поверхности

Подъем стопы при переносе ноги

Контроль высоты тела робота

Управление руками для балансировки

5. Управление:
Через последовательный порт (UART) доступны команды:

r - сброс

g - начать движение

t, y - отладочный режим

Регулировка параметров через числовые команды

6. Безопасность:
detAng() - обнаружение падения и отключение сервоприводов

Ограничения углов суставов

Плавное движение сервоприводов
